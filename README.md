# 铅神经毒性的单细胞研究项目
# Lead Neurotoxicity Single-Cell Analysis

## 项目背景

### 现有研究的局限
- **Bulk组织分析**: 无法区分细胞类型特异性效应
- **单细胞研究**: 目前仅1项 (Bakulski 2020)
  - 仅研究: 内皮细胞(27%), 小胶质细胞(27%), 少突胶质细胞(15%)
  - **未研究**: 神经元、星形胶质细胞

### 创新点
1. **单细胞水平**: 首次全面分析铅对不同脑细胞类型的特异性效应
2. **细胞亚型**: 区分细胞亚型（如小胶质细胞的激活状态）
3. **肿瘤机制借鉴**: 从肿瘤学研究中寻找新机制灵感

---

## 研究问题

1. 铅暴露如何影响不同脑细胞类型的基因表达？
2. 哪些细胞类型对铅最敏感？
3. 铅是否通过肿瘤样机制（如代谢重编程、免疫逃逸）导致神经损伤？

## 分析方法

### 1. 细胞类型分析
```
- 神经元 (Neuron)
- 星形胶质细胞 (Astrocyte)  
- 小胶质细胞 (Microglia)
- 少突胶质细胞 (Oligodendrocyte)
- 内皮细胞 (Endothelial)
- 血管周围细胞 (Pericyte)
```

### 2. 细胞亚型分析
- 小胶质细胞: M1/M2激活状态
- 星形胶质细胞: A1/A3反应状态
- 少突胶质细胞: OL前体/成熟

### 3. 从肿瘤研究借鉴的机制

| 肿瘤机制 | 对应神经损伤机制 |
|---------|----------------|
| 代谢重编程 (Warburg效应) | 铅抑制线粒体功能 |
| 免疫逃逸 | 神经炎症 |
| 血管生成 | 血脑屏障损伤 |
| 表观遗传调控 | DNA甲基化改变 |
| 细胞衰老 | 神经衰老 |

## 技术路线

```
1. 获取公开脑细胞单细胞数据
       ↓
2. 基于铅暴露差异表达基因映射到不同细胞类型
       ↓
3. 细胞类型特异性通路分析
       ↓
4. 细胞间通讯分析 (Cell-cell interaction)
       ↓
5. 借鉴肿瘤机制进行深度分析
       ↓
6. 验证与实验设计
```

## 预期成果

1. **细胞类型特异性标志物**: 识别铅暴露的细胞类型特异性基因
2. **新机制假设**: 提出铅神经毒性的新机制（如代谢重编程）
3. **潜在靶点**: 为治疗干预提供新靶点
4. **论文**: 高水平期刊
